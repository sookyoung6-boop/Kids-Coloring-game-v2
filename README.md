# Coloring Game

Static browser game. No build, no npm, no server.

## GitHub Upload

Upload the contents of `GITHUB_UPLOAD_READY` to the root of a new GitHub repository.

The repository root must contain:

- `index.html`
- `style.css`
- `game.js`
- `assets/`
- `.nojekyll`

Then enable GitHub Pages with:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

Do not upload `CODEX_GPT`, `.github`, `assets/pages`, `assets/background.png`, or `assets/selection-background.png`.
